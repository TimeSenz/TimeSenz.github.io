Ubuntu下安装ROS

ROS安装步骤

稳定版的ROS是寄生在Linux系统下的（测试版也有安装在Android平台和OSX平台的，这里暂不做详细描述），所以在安装ROS系统之前要求有一个Linux操作系统。
而对ROS兼容性最好的当属Ubuntu操作系统，当然，并不是所有版本的ROS都兼容Ubuntu的各个版本。这些版本还很有脾气的互不兼容。所以每一个版本的ROS都对应这一个或多个相对于的Ubuntu版本。

下面是Ubuntu和ROS相对应的版本列表

ROS版本 Ubuntu版本
ROS Kinetic Kame Ubuntu16.04(Xenial)/Ubuntu 15.10(Wily)
ROS Jade Turtle Ubuntu 15.04 (Wily) / Ubuntu LTS 14.04 (Trusty)
ROS Indigo Igloo Ubuntu 14.04 (Trusty)
ROS Hydro Medusa Ubuntu 12.04 LTS (Precise)
ROS Groovy Galapagos Ubuntu 12.04 (Precise)

ROS节点解析

理解ROS节点

先决条件

用你使用的ros发行版本名称（例如electric、 fuerte、 groovy、hydro等）替换掉’‘

图概念概述

  • Nodes: 节点，一个节点即为一种可执行文件，他可以通过ROS与其它节点进行通信
  • Messages：消息，消息是一种ROS数据类型，用于订阅或发布到一个话题
  • Topics： 话题，节点可以发布消息到话题，也可以订阅话题以接收消息
  • Master： 节点管理器，ROS名称服务(比如帮助姐节点找到彼此)
  • rosout： ros中相当于stdout/stderr
  • roscore: 主机 + rosout + 参数服务器

节点

一个节点其实只不过是ROS程序包中的一个可执行文件，ROS节点可以使用ROS客户库于其他节点通信，节点可以发布或接受一个话题。姐弟那也可以提供或使用某种服务。
节点是ros中非常重要的一个概念，为了帮助初学者理解这个概念，这里举一个通俗的例子
例如，咱们有一个机器人和一个遥控器，那么机器人和遥控器开始工作后，就是两个节点。遥控器起到了下达指令的作用。机器人负责监听遥控器下达的指令。完成相应动作。从这里我们就可以看出。节点是一个能执行待定工作任务的工作但愿，并且能够相互通信。从而实现一个机器人整体的功能。在这里我们把遥控器和机器人呢简单定义为两个节点。这个是根据用户编写的程序来定义的

客户端库

ROS客户端库允许使用不同编程语言编写的节点之间互相通信

  • rospy = python客户端库
  • roscpp = C++客户端库
